Ethical Implications (with Ethical Reasoning Framework)

What are the ethical implications when using a self hosted cloud?

The central problem of utilizing ownCloud as our cloud storage solution is that ownCloud is an open-source. Open source means that its source code is available online and this could create a risk for ethical problems to arise. As the source code is accessible at the developers’ fingertips, it potentially allowed a situation where the developers recognizing the bugs residing in the source code but not reporting it to the maintainers of the source code, failing to comply with ACM Code of Ethics (1.3: Be honest and trustworthy).

Such behavior may be deemed to be risky for consumers whose data is in being stored under an open-source platform because the developers that successfully identified the bugs may exploit it to conduct unethical activities such as breaching online users’ private data, violating (1.6 Respect privacy), resulting the data to be insecure. As the core purpose of building private cloud storage is to reinforce the security aspects of managing data, such possible exploitation contradicts the intention of a private cloud.
