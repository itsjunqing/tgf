Tutorial

Learn how to set up your own virtual cloud.

This site provides a comprehensive guide on the step-by-step process for hosting a private cloud storage service. Here, we will be using DigitalOcean as our VPS hoster and ownCloud as our cloud storage.

The steps for setting up have been divided into three parts:

Step 1. Setting Up Virtual Private Server

Before setting up our ownCloud service, we need to have a server that allows us to upload our files to. With this server, it gives us the accessibilty to access our files anywhere with an internet connection.

VPS Setup Guide
Step 2. Setting up LAMP Stack

LAMP stack stands for Linux, Apache, MySQL and PHP. Linux is to provide the operating system for the server, Apache is acting as a web server, MySQL acting as the database and PHP for web interface. In order for ownCloud to function completely, we need to have a LAMP stack.

LAMP Setup Guide
Step 3. Setting Up ownCloud

ownCloud is an open-source file sharing server with collaboration feature that allows you to store your personal files on the cloud. This gives you the accessbility over the files online at anywhere in the world.

As ownCloud is an open-source, this gives you the control over your content and security and to avoid using third-party cloud storage services such as Dropbox or GoogleDrive.
