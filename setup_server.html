Setting Up Virtual Private Server

A virtual private server is a virtual machine sold by an Internet hosting service. It is like a remote computer that runs its own copy of operating system that runs based on 24/7 service. There are many host services across the globe such as DigitalOcean, Microsoft Azure Virtual Machines, or IBM Cloud. Here, DigitalOcean is used as the VPS for hosting a private cloud for demonstration.

# Creating A Project in DigitalOcean

A Project is a space that allows user to manage different virtual resources. Create a project space for a virtual private cloud.

# Creating A Droplet

A Droplet is a Linux-based virtual machines that run according to a cloud-based infrastructure. Create a droplet to host a Linux server.

# Selecting an OS Distribution

This is to install a copy of an operating system onto the virtual machine. There are various Linux OS Distributions that are usable for cloud computing. Here, an Ubuntu server is selected as part of the demonstration.

# Selecting a Droplet Plan

Selecting the droplet plan depends on your needs on private cloud. Large organizations that continuously store large amount of data have a higher tendency to go after a dedicated CPU because it is more stable, as the resources are not shared across multiple users.

On the other hand, smaller organizations would prefer a shared CPU because the needs are not huge. Therefore, choosing the droplet plan would depend on how much data will be stored on the private cloud.

For general purposes, it is recommended to go after the standard version. The difference between a shared CPU and a dedicated CPU can be found under the article DigitalOcean Choosing A Plan.

# Choosing A Datacenter Region

A datacenter is a centralized location where computing and networking equipments are concentrated for the purpose of managing large amount of data. It can be considered as a center with huge amount of computers. Data centers are typically built throughout the world to provide a scalable and efficient service to people across the world.

Choosing the right datacenter will effectively boost data speeds and latency transfers. It is recommended to pick a data center that is closest to your region. (Singapore is closest to us)

# Choosing Droplet Authentication Method

This is to verify the identity. SSH or Secure Shell is a network protocol, or a method of communicating between two machines remotely with a strong encryption algorithm, allowing one person to gain control of the servers via the internet.

SSH approach creates better security than passwords due to the encryption approach used, but usage ultimately boils down to the user's needs. For demostration purposes, we will be using the one-time password approach.

# Enabling A System-level Backup (Optional)

An optional choice of backing up the server weekly where the backup data is stored for a period of at most one month. For more information, refer to DigitalOcean Backup Overview.
